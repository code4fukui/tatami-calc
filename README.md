# tatami-calc
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple calculator for determining the size and number of tatami mats needed for a room.

## Demo
https://code4fukui.github.io/tatami-calc/

## Features
- Calculate the dimensions of a room based on the size and number of tatami mats
- Determine the optimal number and size of new tatami mats to fit the room
- Display the difference in dimensions between the original and new mats

## Usage
1. Enter the size of the tatami mats (width and length)
2. Select the number of tatami mats for the current room layout
3. The calculated room dimensions will be displayed
4. Select the desired number of new tatami mats
5. The dimensions of the new tatami layout and the difference from the original will be shown
