# tatami-calc
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

畳マットの必要な大きさと数を計算する簡単な計算機。

## デモ
https://code4fukui.github.io/tatami-calc/

## 機能
- 畳の数と大きさに基づいて部屋の寸法を計算する
- 部屋に合うよう最適な新しい畳の数と大きさを決定する
- 元の畳と新しい畳の寸法の違いを表示する

## 使用方法
1. 畳の大きさ(幅と長さ)を入力してください
2. 現在の部屋レイアウトの畳の数を選択してください
3. 計算された部屋の寸法が表示されます
4. 新しい畳の数を選択してください
5. 新しい畳のレイアウトの寸法と、元の寸法との差が表示されます
